# 📊 Student Performance EDA

## 🎯 Project Objective
Identify behavioral and lifestyle factors affecting student exam performance using data analytics.

## 🛠️ Tech Stack
- Python · Pandas · NumPy
- Matplotlib · Seaborn · Plotly
- Data Cleaning · Feature Engineering · KPI Analysis

## 📌 Key Findings
- ✅ Study hours = #1 driver of exam performance
- 😴 Good sleep (7–9 hrs) = ~64% higher scores
- 📱 Social media usage reduces productivity
- ☕ Caffeine has minimal impact on performance
